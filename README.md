# My Dotfiles

These are my personal dotfiles for my Arch + Hyprland setup, Big shoutout to [mikkurogue](https://github.com/mikkurogue/) for the waybar config, I've just added some bits

## Structure

```
.dotfiles/
├── .config/
│   ├── hypr/         # Hyprland config
│   ├── waybar/       # Status bar
│   ├── albert/       # App launcher
│   ├── ghostty/      # Terminal
│   ├── starship.toml # Shell prompt
│   ├── nvim/         # Neovim (LazyVim)
│   ├── dunst/        # Notifications
│   └── fastfetch/    # System fetch
```

## Setup

1. Clone this repo to `~/dotfiles`
2. Symlink or copy configs to `~/.config/`
3. Install dependencies

## Dependencies

### Core
- hyprland (wayland compositor)
- ghostty (terminal)
- waybar (status bar)
- albert (app launcher)
- starship (shell prompt)
- neovim (editor)
- dunst (notifications)
- fastfetch (system info)

### Fonts
- CaskaydiaCove Nerd Font

## Excluded from repo

- Game configs (Heroic, RetroArch, Paradox Launcher)
- Large apps (Discord, OBS, Spotify)
- Caches and app data
